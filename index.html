Start or Sit

Start or Sit, is a fantasy football application used to help players collaborate and make decisions that could benefit his or her own fantasy football team. Start or Sit includes a voting system in which users can vote on the player they think should start, a comment system for players to voice their opinions, live scores, and in-depth player news and statistics.

Technologies Used: AngularJS, Angular-Router, jQuery, Grunt, Bower, Sass, HTML5, Bootstrap, Materialize, and Firebase

Popcorn Classics

Popcorn Classics, is a movie saving and rating application. Users are able to search through a movie database, and save their selected movies to their unwatched list. After the movie has been watched, the user is then able to move the movie to their watched list and then rate it on a 0 to 10 scale. Users can then filter their movies based on its rating for future reference.

Technologies Used: JavaScript, jQuery, Bower, Grunt, Materialize, Bootstrap, Firebase, HTML5, and Sass

FriendBook

FriendBook, is a Facebook clone that allows user to create a user account or login. After logging in users then have the ability to search and send friend requests to other users, post new statuses, like dislike and comment on those statuses, a conversation system that allows users to message each other, message notifications, post notifications, friend request notifications, image uploads, photo albums, a color customizable profile page and home page, and much more.

Technologies Used: C#, .NET Core, jQuery, SQL, Linq, and Razor
